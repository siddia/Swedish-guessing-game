# Swedish Furniture Name

A one-page quiz game where players guess what Swedish product names actually are.

## Features
- 10 rounds per game
- 4 answer choices for each round
- JSON-driven questions
- Running score and progress bar
- Fun facts after each answer
- Confetti celebration for correct answers
- Final results screen with a Try Again button

## Files
- `index.html`
- `style.css`
- `script.js`
- `questions.json`

## Run locally
Use any local server, for example:

```bash
python3 -m http.server
```

Then open the local address in your browser.

## Deploy to GitHub Pages
1. Create a new repository named `guess-the-furniture-name`
2. Upload these files to the repo root
3. Go to **Settings → Pages**
4. Under **Build and deployment**, choose **Deploy from a branch**
5. Select the `main` branch and `/root`
6. Save, then wait for GitHub Pages to publish your site
